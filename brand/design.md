# Buddy — Charte graphique

## Couleurs

| Nom         | Hex       | Rôle                       |
|-------------|-----------|----------------------------|
| Orange      | `#FF5F40` | CTA, accent principal      |
| Vert foncé  | `#23554C` | Fond sombre, sections dark |
| Bleu clair  | `#ABD4FD` | Accent secondaire          |
| Jaune clair | `#FCF3A4` | Highlight, tags            |
| Vert lime   | `#AAC538` | Succès, badges actifs      |
| Rose        | `#FD9EBC` | Tags, accents doux         |
| Fond        | `#FFFDD0` | Fond principal             |
| Texte       | `#4D3326` | Texte principal            |

## Typographies

| Usage  | Police    |
|--------|-----------|
| Titres | Gelica    |
| Corps  | Helvetica |

## CSS Variables

```css
:root {
  --orange: #FF5F40;
  --vert:   #23554C;
  --bleu:   #ABD4FD;
  --jaune:  #FCF3A4;
  --lime:   #AAC538;
  --rose:   #FD9EBC;
  --fond:   #FFFDD0;
  --texte:  #4D3326;
}
```
